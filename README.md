# AgroGestión 🐄
**Sistema Integral de Gestión Ganadera (Ceba y Levante)**

Plataforma profesional para el control de inventario, pesajes, KPI zootécnicos y seguimiento de salud animal en fincas ganaderas colombianas.

## ✨ Características Principales
- **Dashboard en Tiempo Real**: Visualización de GDP, GMP y mortalidad anual.
- **Control de Inventario**: Gestión por chapetas, propietarios y etapas.
- **Registro de Pesajes**: Cálculo automático de ganancia de peso y alertas de rendimiento.
- **Multi-Finca**: Soporte para administrar múltiples predios con un solo usuario.
- **Diseño Responsive**: Optimizado para uso en campo desde dispositivos móviles.

## 🚀 Tecnologías
- **Frontend**: React + TypeScript + Vite.
- **Estilos**: Vainilla CSS con enfoque Glassmorphism.
- **Backend**: Supabase (Base de Datos + Auth).
- **Iconografía**: Lucide React.
